## ⚡ Quick States Reference

| State | Icon | Appearance |
|-------|------|------------|
| **Unselected** | ☐ | Gray checkbox, white background |
| **Selected** | ☑ | Blue checkbox, light blue background |
| **Hover** | → | Light gray background |
| **Focus** | ⚡ | Blue border/ring |
| **Loading** | ⏳ | Spinner, gray text |
| **Error** | ❌ | Red text, red border |
| **Empty** | ∅ | Gray text, centered |
| **Search Active** | 🔍 | Blue icon, focused input |

## 🎯 Interactive Elements

### Click Targets

```
┌─────────────────────────────────────────────┐
│ Campaign Name                               │  ← Label (44px height)
├─────────────────────────────────────────────┤
│ Select Campaign Name              ▼         │  ← Trigger (44px height)
└─────────────────────────────────────────────┘
  ↑ Entire area clickable

┌─────────────────────────────────────────────┐
│ ☐ Campaign A                               │  ← Item (44px height)
└─────────────────────────────────────────────┘
  ↑ Entire row clickable

[Badge Text ✕]  ← Badge (32px height, ✕ is 24px clickable)
```

## 📐 Dimensions

### Desktop

```
Width:  100% (parent container)
Height: 40px (trigger)
        up to 300px (dropdown list)
Padding: 12px (trigger)
         8px (items)
Font Size: 14px (normal)
           16px (label)
```

### Mobile

```
Width:  100% (viewport width - padding)
Height: 44px (trigger) - Touch-friendly
        up to 60vh (dropdown list)
Padding: 12px (trigger)
         12px (items) - Larger for touch
Font Size: 16px (prevent zoom on iOS)
```

## 🎨 Animation

### Open Animation

```
Frame 1: Closed
┌─────────────────┐
│ Select...    ▼  │
└─────────────────┘

Frame 2: Opening (opacity 0 → 1, scale 0.95 → 1)
┌─────────────────┐
│ Select...    ▲  │
├─────────────────┤
│ ┌─────────────┐ │
│ │ Option A    │ │ ← Fading in
│ └─────────────┘ │

Frame 3: Open
┌─────────────────┐
│ Select...    ▲  │
├─────────────────┤
│ ┌─────────────┐ │
│ │ Option A    │ │
│ │ Option B    │ │
│ │ Option C    │ │
│ └─────────────┘ │
```

### Select Animation

```
Before Select:
☐ Option A

During Click (scale 0.95):
☑ Option A

After Select (scale 1):
☑ Option A
```
