TitleComponent Structure
========================

Component Hierarchy:
====================

<div> (Container with styling)
  │
  ├─ Props: backgroundColor, border, borderRadius, padding, shadow
  ├─ Hover effects (when editable)
  ├─ Edit mode ring indicator
  │
  └─── <h1|h2|h3|h4|h5|h6> (Dynamic heading tag)
       │
       ├─ Props: fontFamily, fontSize, fontWeight, color, alignment
       ├─ contentEditable (when editing)
       ├─ Focus styles
       ├─ Text content or placeholder
       │
       └─── [Edit Hint] (absolute positioned, top-right)
            └─ "Click to edit" (visible on hover)
       
       └─── [Editing Hint] (absolute positioned, bottom)
            └─ "Press Enter to save, Esc to cancel"

State Flow:
===========

Initial State
    │
    ├─ text: "Add title here..."
    ├─ isEditing: false
    └─ originalText: "Add title here..."

User Clicks
    │
    ├─ setIsEditing(true)
    ├─ Focus and select all text
    └─ Show editing hint

User Edits
    │
    └─ setCurrentText(newValue)

User Saves (Enter or Click Outside)
    │
    ├─ setIsEditing(false)
    ├─ Trim and validate text
    ├─ onTextChange(finalText)
    └─ setOriginalText(finalText)

User Cancels (Escape)
    │
    ├─ setIsEditing(false)
    └─ Revert to originalText

Props Flow:
===========

ComponentConfig (Dashboard State)
    │
    ├─ id: "title-1"
    ├─ type: "title"
    ├─ text: "Dashboard Overview"
    ├─ headingLevel: "h1"
    ├─ fontSize: "32"
    ├─ fontWeight: "700"
    ├─ color: "#1A73E8"
    └─ alignment: "center"
    │
    └─> TitleComponent Props
         │
         └─> Rendered Styles
              │
              └─> DOM Elements

Event Handlers:
===============

onClick (Container)
    └─> handleClick()
        └─> setIsEditing(true)

onInput (Heading)
    └─> handleInput()
        └─> setCurrentText(newValue)

onKeyDown (Heading)
    ├─> Enter → handleSave()
    └─> Escape → handleCancel()

Click Outside
    └─> useEffect cleanup
        └─> handleSave()

Styling System:
===============

Container (div)
    │
    ├── CSS-in-JS (Dynamic)
    │   ├─ backgroundColor
    │   ├─ border
    │   ├─ borderRadius
    │   ├─ padding
    │   └─ boxShadow
    │
    └── Tailwind (Static)
        ├─ w-full h-full
        ├─ transition-all
        ├─ hover:ring-2 (when editable)
        └─ ring-2 (when editing)

Heading Tag
    │
    ├── CSS-in-JS (Dynamic)
    │   ├─ fontFamily
    │   ├─ fontSize
    │   ├─ fontWeight
    │   ├─ color
    │   ├─ textAlign
    │   └─ cursor
    │
    └── Tailwind (Static)
        ├─ focus:outline-none
        └─ text-muted-foreground (placeholder)

Accessibility Tree:
===================

When Editable:
    heading (h1-h6)
        ├─ role: textbox
        ├─ aria-label: "Editable title"
        ├─ aria-multiline: false
        └─ contenteditable: true/false

When Not Editable:
    heading (h1-h6)
        ├─ role: heading
        ├─ aria-level: 1-6
        └─ text content

Keyboard Navigation:
====================

Tab
    └─> Focus heading (if editable)

Click
    └─> Enter edit mode

Enter
    └─> Save and exit

Escape
    └─> Cancel and revert

Tab (while editing)
    └─> Save and move focus
