## 10. COMPONENT SPECIFICATIONS

### All UI Elements with Icons (Lucide React):

**Topbar Icons:**
- Menu: `Menu` icon
- Undo: `Undo` icon
- Redo: `Redo` icon
- Cursor: `MousePointer` icon
- Add Row: `Plus` + `Rows` icons
- Add Chart: `BarChart3` icon
- Add Control: `SlidersHorizontal` icon
- Agent: `Bot` or `Sparkles` icon
- Share: `Share2` icon
- View: `Eye` icon
- Save: `Save` or `Cloud` icon
- User: `User` or avatar image

**Sidebar Icons:**
- Close: `X` icon
- Chart type: Varies by chart
- Data source: `Database` icon
- Dimension: `Tag` icon (green badge)
- Metric: `TrendingUp` icon (blue badge)
- Filter: `Filter` icon
- Date: `Calendar` icon
- Settings: `Settings` icon
- Drag handle: `GripVertical` icon
- Remove: `X` icon
- Sort: `ArrowUpDown` icon
- Compare: `Scale` icon
- Add: `Plus` icon

**Row/Column Icons:**
- Drag handle: `GripVertical`
- Delete row: `Trash2`
- Delete component: `X`
- Settings: `Settings`
- Copy: `Copy`
